# Usage

> Make sure kiqpo is installed


```console
desktop@kiqpo:~$ kiqpo
v1.0.0
```

### cerate new kiqpo app 
This will cerate new kiqpo app
```console
kiqpo cerate my-app
```

<br/>

## Runing

 > runing on web

This will serve local page
```console
kiqpo run-web
```


<br/>

 > runing on desktop

This start a desktop app
```console
kiqpo run-desktop
```
