# Usage
Make sure kiqpo is installed


```console
desktop@kiqpo:~$ kiqpo
v1.0.0
```

## Cerate new kiqpo app 
This will cerate new kiqpo app
```console
kiqpo cerate my-app
```

<br/>

## Runing


This will serve local page
```console
kiqpo run-web
```


<br/>

This start a desktop app
```console
kiqpo run-desktop
```
