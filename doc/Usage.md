### Usage
Make sure kiqpo is installed


```console
desktop@kiqpo:~$ kiqpo
v1.0.0
```


# 


### Cerate new kiqpo app 
<details>
  
  <summary>Click to expand!</summary>
 
  <sub>This will cerate new kiqpo app with name my-app</sub>
  
```console
kiqpo cerate my-app
```
</details>
 

### Runing
<details>
  
  <summary>Click to expand!</summary>
  

  <sub>This will serve local page</sub>
  
  ```console
kiqpo run-web
```
</details>



### Runing - desktop
<details>
  
  <summary>Click to expand!</summary>
  
  <sub>This start a desktop</sub>
  
  ```console

```
</details>
